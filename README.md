What is virtenviro
==================

Open source content management system (CMS) based on the django framework.

* it's simple to start
* it's simple to use
* it's simple to extend

Packages required
=================


* 'Django 1.7.*',
* 'django-mptt',
* 'pytils',
* 'lxml',
* 'Pillow',
* 'sorl-thumbnail',
* 'django-filebrowser-no-grappelli',
